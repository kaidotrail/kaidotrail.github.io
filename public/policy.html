旧街道足跡マップ (以下「本サイト」) は、個人情報の保護に関する各種法令を遵守し、プライバシーポリシーに基づいて個人情報の適切な取り扱いを行います。 改定された場合は、本ページの改定履歴欄でお知らせします。

投稿フォーム・連絡フォームにおける連絡先情報の取り扱い

本サイトの投稿フォームや連絡フォームに記入されたメールアドレスは、連絡に対する回答や必要な情報を返信するためのみ使用し、それ以外では使用しません。 ハンドルネーム・X や GitHub 等のアカウント名は、連絡に対する回答や必要な情報を返信するために利用するほか、投稿内容がサイトに反映されるものであった場合はクレジットとして掲載されます。 ただし、クレジット掲載を望まない旨記載があった場合はこの限りではありません。

フォームには個人を特定できる情報は記入しないよう案内していますが、万が一記入された場合は投稿データ一式を永続的に破棄し、本サイトでは取り扱わないものとします。 また、利用者から送信内容のデータの破棄の申し出があった場合も同様に対応します。

暗号化の有無とリンク先

本サイトの内容は原則として全ての接続が SSL/TLS によって暗号化されています。 ただしリンク先のウェブサイト等においては、学術的に有用な情報源・引用元であることも鑑み、暗号化されていないページであっても紹介しています。 非暗号化サイトの接続に問題がある場合は、ブラウザの設定等で予め対策いただくようお願いします。

外部ドメインへの接続

本サイトは、地図タイル・フォント・JavaScript ライブラリの取得やアクセス分析サービスの利用のため、ブラウザに表示されているドメイン以外の外部ドメインと通信します。 外部ドメインは本サイトと異なるポリシーで運営されており、IP アドレスや利用環境などが収集される可能性があります。 なお、これらは匿名で収集されており、個人を特定するものではありません。 全ての外部ドメインの運営元の一覧は以下の通りです:

  • Google – 地図タイルの取得、Web フォントの取得、Tag Manager によるタグ挿入、Analytics へのデータ送信 (後述)、広告配信 (後述) のため
  • Cloudflare – JavaScript ライブラリの取得、アイコンフォントの取得、Web Analytics へのデータ送信 (後述) のため
  • 国土地理院 – 地図タイルの取得のため
  • Esri – 地図タイルの取得のため
  • 今昔マップ on the web – 地図タイルの取得のため
  • 農業環境技術研究所 – 地図タイルの取得のため
  • Creative Commons – ライセンスバッジ取得のため
  • jsDelivr – JavaScript ライブラリの取得のため
  • Formspree – 連絡フォーム・投稿フォームの受付のため
  • imgBB – 投稿フォームの画像の受付のため

本サイトは GitHub によりホスティングされています。GitHub のポリシーは https://docs.github.com/site-policy を参照してください。

アクセス分析

本サイトでは Google Analytics と Cloudflare Web Analytics を用いてページのアクセス回数、流入経路、利用環境、表示速度の測定結果などを収集し、コンテンツの改善や表示速度の改善のための分析に活用しています。

Google Analytics は Cookie や広告識別子を利用して行動分析を行います。Google のポリシーは https://policies.google.com/technologies/partner-sites を参照してください。 Google Analytics をオプトアウトするには https://tools.google.com/dlpage/gaoptout/ を参照してください。

Cloudflare Web Analytics は Cookie・localStorage・フィンガープリンティングを用いずに行動分析を行うサービスです。Cloudflare のプライバシーポリシーは https://www.cloudflare.com/privacypolicy/ を参照してください。Cloudflare Web Analytics に公式のオプトアウト手段はありませんが、一部の広告ブロッカーが Web Analytics の遮断に対応しています。

広告

当サイトでは広告配信サービスとして Google AdSense を利用しており、ユーザーの興味に応じた商品やサービスの広告を表示するため Cookie を使用しています。詳細は https://policies.google.com/technologies/ads を参照してください。 また、本サイトは、Amazon.co.jpを宣伝しリンクすることによってサイトが紹介料を獲得できる手段を提供することを目的に設定されたアフィリエイトプログラムである、Amazonアソシエイト・プログラムの参加者です。
